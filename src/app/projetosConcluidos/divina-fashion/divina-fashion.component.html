Projeto Divina Fashion

Este projeto foi o primeiro projeto desenvolvido em um curso de formação angular para melhorar a habilidade com:

  • HTML
  • CSS
  • Responsividade
    • Desktop
    • Tablet
    • Mobile

O layout, cores e posições dos elementos deste projeto, foi feito seguindo exatamente um Figma disponibilizado pelo curso.
