Projeto LocCar Dialog reserva carro usuário comum

Ao clicar na categorias que deseja o usuário irá ter um dialog para observar quantas portas o carro tem, quantas pessoas cabem no veículo e o nome da locadora do carro selecionado.

Se caso o usuário escolher reservar o carro selecionado, o usuario vai ser direcionado irá para a parte de reservas para preencher data da reserva, a hora da retirada e a data de devolução.

Projeto LocCar Página reserva usuário comum

Fiz um cadastro de um carro para melhorar a visualização.
Todos os carros cadastrados aparecem no campo Minhas reservas.
Cada card de carro cadastrado tem um botão editar reserva do carro e um botão excluir reserva do carro.

Projeto LocCar Dialog excluir reserva de carro usúario comum

Cada card tem um dialog caso o usuário desejar excluir ou editar uma reserva ja cadastrada.

Aqui acaba o fluxo de um usuário comum da plataforma.
Apartir daqui as imagens irão ser de um usuário ADM é o mesmo fluxo porém tem algumas funcionalidades extra como voce pode ver logo abaixo:

Projeto LocCar Página perfil administrador

Em uma conta ADM é adicionada Locadoras na barra de navegação.
O perfil de um ADM lista todos os usuários cadastrados na plataforma. Caso o ADM quiser editar um usuário ele dá um click no card do usuário e os dados do usuário sobem para o input para serem atualizados.
O ADM pode também excluir um usuário clicando no botão excluir do card do usuário.
Se o ADM clicar no excluir vai abrir um dialog para ele ter certeza se deseja excluir o usuário.

Projeto LocCar Página carros administrador

Na página de carros de um ADM tem como o ADM cadastrar carros disponíveis nas locadoras, que aparecerão na página do usuário comum, para realizar a reserva.
Em cada card de carros cadastrados tem um botão de editar e excluir .

Projeto LocCar Dialog editar carro

Ao clicar em editar um carro, um dialog aparece para a atualização do nome, tipo do carro (popular, sedan ou luxo), números de portas, capacidade de pessoas do carro e a locadora que o carro se encontra.

Projeto LocCar Dialog excluir carro

Ao clicar em excluir em um carro selecionado, aparece um dialog com a mensagem se deseja realmente excluir o carro selecionado.

Projeto LocCar Página reserva administrador

Na página reserva de um ADM vai listar todos os carros que estão reservados, para atualizar alguma informação do veiculo o ADM clica no card da reserva e os dados vão para os inputs para serem atualizados.
O ADM tambem tem como excluir uma reserva cadastrada, mesmo procedimento..dialog e se realmente deseja excluir a reserva.

Projeto LocCar Página cadastrar / editar locadora de veículo

Na página exclusiva de ADM locadora, o ADM é capaz de cadastrar uma nova locadora, editar uma existente ou excluir locadora.
Caso o ADM cadastrar uma locadora de veículo nova, vai aparecer no select da pagina carros, para cadastrar carros da nova locadora.

Projeto LocCar Dialog editar locadora de veiculo

Ao clicar em Editar de um card de uma locadora cadastrada, aparece um dialog para atualizar dados da locadora selecionada.
Aqui termina o fluxo de um ADM na plataforma.
