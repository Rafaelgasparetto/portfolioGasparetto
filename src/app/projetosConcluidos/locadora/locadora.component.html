Projeto Filme Collection Pagina Home

Está é a pagina home da locadora com header/footer como component e a pagina com sistemas de routes - router-outlet.
Foi usado Mat-card do angular material para fazer o card Filmes e categorias. Ali aonde está o logo Filmes colletion, tem uma mensagem "conheça nossa coletânea" ao cada pagina ela vai se alterando como pode ver nas proximas imagens.

Projeto Filme Collection Pagina Cadastro de usuários

Os campos do form são com mat-label / matInput estilizados de acordo com o figma.
Nos campos como podem ver, esta com os validators e quando todos os campos estiverem válidos o botão fica habilitado para prosseguir o cadastro de usuário.

OBS: no header para trocar de mensagem a cada pagina utilizei o *ngIf

Projeto Filme Collection Dialog do editar usuários

Em cada card do usuário tem o botão de editar, e quando o usuário clica nele aparece um dialog para fazer a alteração do usuário selecionado, com o botão de atualizar ou cancelar.

OBS: O dialog não foi exigido no desafio, eu achei importante ter o dialog para o usuário pois facilita a visualização do mesmo, a estilização do dialog foi por minha conta, seguindo os estilos e cores da pagina.

Projeto Filme Collection Pagina cadastro de gêneros

Nesta pagina, como a pagina de cadastro de usuario tem validators e desabilita o botão quando o campo for valido.
Achei importante trazer esse fluxo de apresentação pois na parte de filmes tem um campo de select que vai mostrar os generos cadastrados em um select para cadastrar o filme.

Projeto Filme Collection Dialog editar gêneros

Ao clicar em editar do gênero aparece um dialog para fazer a alteração com validators e required, junto com dois botões de cancelar e editar (o botão editar está bloqueado porque o campo gênero esta inválido).
