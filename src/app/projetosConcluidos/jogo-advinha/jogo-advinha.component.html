Projeto OJOGO Inicio do Game

Como mostra no retângulo vermelho, o usuário coloca o nick.
No retângulo azul seleciona o intervalo, possui 3 niveis de opções
Assim que os dois campos forem selecionados o nick do usuário apareçe para dar uma boas vindas ao usuário.
Ao clicar em começar o programa escolhe um número randômico de acordo com a opção escolhida pelo usuário.

Projeto OJOGO Iniciado o Game

No retângulo em vermelho esta o número que o usuário escolheu, assim que ele clicar em jogar o numero que ele colocou será comparado ao numero secreto randômico, nesta imagem o numero que foi escolhido é menor que o numero gerado, como mostra no retângulo em azul.
Ainda no retângulo azul o usuário consegue ver quantas tentativas ele possui.

Projeto OJOGO Game Over

Caso o usuário perder todas as chances que teve para acertar o numero, o jogo vai mostrar o numero randômico gerado no começo do jogo, exibirá uma mensagem para o usuário como esta do retângulo vermelho

Projeto OJOGO Win

Quando o usuário acerta o palpite, o jogo exibe um texto de um parabéns para o usuário
