Projeto OJOGO Início do Game

Como mostra no retângulo vermelho, o usuário coloca o nick.
No retângulo azul seleciona o intervalo que possui 3 níveis de opções.
Assim que os dois campos forem selecionados, o nick do usuário aparecerá para dar as boas-vindas ao usuário.
Ao clicar em começar, o programa escolhe um número randômico de acordo com a opção escolhida pelo usuário.

Projeto OJOGO Iniciado o game

No retângulo em vermelho está o número que o usuário escolheu, assim que ele clicar em jogar o número que ele colocou será comparado ao número secreto randômico, nesta imagem o número que foi escolhido é menor que o número gerado, como mostra no retângulo em azul.
Ainda no retângulo azul o usuário consegue ver quantas tentativas ele possui.

Projeto OJOGO Game Over

Caso o usuário perca todas as chances que teve para acertar o número, o jogo vai mostrar o número randômico gerado no começo do jogo. Exibirá uma mensagem para o usuário como está do retângulo vermelho.

Projeto OJOGO Win

Quando o usuário acerta o palpite, o jogo exibe um texto de parabéns para o usuário.
